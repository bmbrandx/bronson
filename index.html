Introduction

Bronson is an opinonated application framework comprising of the tools necessary to build large scalable web applications as outlined by Nicholas Zakas and Addy Osmani. Bronson sit's on top of RequireJS & JQuery offering libraries for the loading, unloading, messaging and bootstraping of modules. These modules in turn acts as small loosely coupled applications in the context of a large appication. By building our larger applications as a series of smaller applications we greatly increase the performance, testability and maintainability of our code base.

Acknowledgements

  • Nicholas Zakas: Scalable Javascript Application Architecture
  • Addy Osmani: Patterns For Large-Scale JavaScript Application Architecture

Features

  • Module loading & unloading through Bronson.Core
  • Univeral interface offered through Bronson.Api
  • Managed publish & subscribing of events through Bronson.Core & Bronson.Permissions

Dependencies

  • jQuery
  • RequireJS
